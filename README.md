## 项目技术
* html+css+javascript
* bootstrap
* vue

## 网站包含
1. 表单（搜索，登陆）
2. 导航栏
3. 侧边栏
4. 一个基本布局（flex,float）
5. 响应式布局
6. 跑马灯
7. cookie
8. 登陆客户端验证

## 项目功能
1. 有趣
  * 当无聊的时候打开网页，就可以发现很多很有趣的网页。
2. 有用
  * 里面收藏了很多有用的网站，例如抠图、PDF转word、录屏等。
3. 学习
  * 可以在里面获取学习的乐趣。里面有收藏学习网站。
4. 收藏
  * 将来可以将自己觉得不错的网站写在上面，起到收藏的作用。
5. 打发时间
  * 里面有很多无聊的小游戏，可以玩一整天。
